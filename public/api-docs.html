M-SM AI Developer API

The M-SM AI API lets you integrate AI marketing content generation directly into your own applications, websites, and workflows. Generate blog posts, ad copy, email campaigns, product descriptions, and more with a simple HTTP request.

✅ Base URL: https://msm-ai-production.up.railway.app

All API requests must be authenticated using an API key. Responses are returned in JSON format.

Authentication

The M-SM AI API uses API keys for authentication. Pass your API key in the X-API-Key header with every request.

curl -X POST https://msm-ai-production.up.railway.app/api/generate \
  -H "X-API-Key: mak_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Write a blog post about AI marketing"}'
⚠️ Keep your API key secret. Do not expose it in client-side JavaScript or public repositories.

API Keys

Generate your API keys from the M-SM AI dashboard under Settings → Developer API. Each account can have up to 5 active API keys.

API keys start with mak_ and are shown only once when created. Store them securely.

Generate Content

POST /api/generate Generate marketing content

Request Body

ParameterTypeRequiredDescription
promptstringRequiredThe content generation prompt
toolIdstringOptionalTool identifier (e.g. blog-writer, ad-copy)
toolNamestringOptionalDisplay name for the document
tonestringOptionalProfessional, Friendly, Bold, Witty, Casual. Default: Professional
variantsintegerOptionalNumber of variants (1-3). Default: 1

Example Request

curl -X POST https://msm-ai-production.up.railway.app/api/generate \
  -H "X-API-Key: mak_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Write a Facebook ad for a fitness app targeting busy professionals",
    "toolId": "ad-copy",
    "toolName": "Ad Copy",
    "tone": "Bold",
    "variants": 2
  }'

Example Response

{
  "success": true,
  "text": "Stop making excuses. Start making gains...",
  "wordCount": 87,
  "creditsUsed": 10,
  "docId": 142,
  "seoScore": 72,
  "readabilityScore": 85
}

AI Chat

POST /api/chat Send a message to the AI marketing assistant

Request Body

ParameterTypeRequiredDescription
messagesarrayRequiredArray of message objects with role and content
modelstringOptionalGroq model ID. Default: llama-3.3-70b-versatile

Example Request

curl -X POST https://msm-ai-production.up.railway.app/api/chat \
  -H "X-API-Key: mak_your_key" \
  -H "Content-Type: application/json" \
  -d '{
    "messages": [
      {"role": "user", "content": "Give me 5 email subject line ideas for a Black Friday sale"}
    ]
  }'

Example Response

{
  "success": true,
  "message": "Here are 5 high-converting subject lines:\n1. ⚡ 60% Off Ends Tonight...",
  "model": "llama-3.3-70b-versatile"
}

Documents

GET /api/documents List all documents

Query Parameters

ParameterTypeDescription
pageintegerPage number. Default: 1
limitintegerResults per page. Default: 30, Max: 100
 
curl https://msm-ai-production.up.railway.app/api/documents?page=1&limit=10 \
  -H "X-API-Key: mak_your_key"
GET /api/documents/:id Get a single document 
curl https://msm-ai-production.up.railway.app/api/documents/142 \
  -H "X-API-Key: mak_your_key"
DELETE /api/documents/:id Delete a document 
curl -X DELETE https://msm-ai-production.up.railway.app/api/documents/142 \
  -H "X-API-Key: mak_your_key"
